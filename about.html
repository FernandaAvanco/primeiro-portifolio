Sobre mim

Sou estudante de Engenharia de Software com objetivo de me tornar uma Desenvolvedora Full Stack. Tenho grande disposição para o aprendizado e aplicação de conhecimentos e habilidades interpessoais no âmbito profissional, buscando crescimento constante.

Meus conhecimentos se baseiam em lógica de programação, linguagens C e C++, HTML5 e CSS3, Engenharia de Requisitos de Softwares, UML, manipulação do terminal do sistema operacional LINUX, manipulação dos editores de código Visual Studio Code e CodeBlocks e inglês fluente. Atualmente estou estudando mais sobre JavaScript, Git e GitHub. Focada nos meus objetivos, pretendo crescer pessoal e profissionalmente, contribuindo também para o crescimento e melhoria da empresa onde trabalharei.
